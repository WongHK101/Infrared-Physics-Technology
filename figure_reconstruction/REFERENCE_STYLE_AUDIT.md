# Reference-style figure audit

Date: 2026-07-22

## Reference set

The visual comparison used the four papers in the local `reference papers` directory,
including the UAV alignment TGRS paper, RoMa, the selective-risk reference, and the
infrared registration journal paper.

## Audit result

- **Evidence density:** the manuscript now uses nine complementary figures rather than a
  small set of isolated plots. The sequence covers the problem, benchmark organization,
  framework, mechanism, pairwise evidence, scene products, selection behavior, operating
  profile, and controlled validation.
- **Scientific hierarchy:** overview figures use real imagery and concise dataset cards;
  method figures expose the actual variables and decisions; result figures connect summary
  statistics to representative alignment products.
- **Visual language:** white background, restrained and consistent colors, sans-serif type,
  direct labels, vector charts, and real image crops match the conventions of the reference
  papers while preserving a coherent manuscript identity.
- **Print readiness:** every figure is available as editable-text SVG, single-page PDF, and
  300-dpi PNG. Critical distinctions also use marker shape, line style, or label text so that
  interpretation does not depend on color alone.
- **Source closure:** all quantitative panels map to frozen CSV/JSON artifacts, all image
  panels use verified local assets, and no active placeholder remains.

## Manuscript placement

Figures 1--2 establish the benchmark and protocol; Figures 3--4 define the framework and
decision mechanism; Figures 5--9 provide pairwise, qualitative, reliability, operating-profile,
and attribution evidence. This ordering is suitable for direct manuscript integration.

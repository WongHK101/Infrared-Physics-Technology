# UAV-TAlign-12K Supplement Skeleton

This directory is a working skeleton for the IP&T supplementary material. It is intentionally lightweight and contains tables, source artifacts, and placeholders needed for manuscript polishing. It does not include raw images, model weights, server credentials, or full experiment output directories.

## Proposed Structure

1. Reliability score design and formula.
2. Threshold sensitivity.
3. Per-scene reliability table.
4. Condition reliability profile.
5. RIFT2 light-vs-strong sanity check.
6. Manifest integrity summary.
7. Additional qualitative examples.
8. SIFT/AKAZE 12K completeness table, once the background run finishes.
